# 📊 Sales Dashboard – Tableau

## 📌 Project Overview

This project is an interactive **Sales Dashboard** built using **Tableau** to analyze sales performance across different regions, categories, and customer segments. The dashboard helps users identify trends, monitor key performance indicators (KPIs), and make data-driven business decisions.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Compare sales across different regions.
* Identify top-performing product categories.
* Monitor profit and sales trends.
* Enable interactive filtering for deeper analysis.

---

## 🛠️ Tools Used

* **Tableau**
* **Sample Superstore Dataset**
* Data Visualization
* Dashboard Design

---

## 📈 Dashboard Features

* ✅ Total Sales KPI
* ✅ Total Profit KPI
* ✅ Sales by Region
* ✅ Sales by Category
* ✅ Profit Analysis
* ✅ Interactive Filters
* ✅ Clean and User-Friendly Dashboard Layout

---

## 📊 Key Insights

* Identified the regions contributing the highest sales.
* Compared category-wise sales performance.
* Highlighted profitable and low-performing segments.
* Enabled interactive exploration using dashboard filters.
